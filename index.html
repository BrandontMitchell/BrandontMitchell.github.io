Work Experience

  • May 2019 - Aug 2019

    Feasible, Inc.

    Software Engineer Intern

    Last summer I worked as a Software Engineer Intern at Feasible, Inc. Feasible is a battery diagnostics company, centered around the idea that in order for there to be a sustainable future, we need to analyze batteries more efficiently (cost and production). Through their ultrasonic testing and machine learning analysis, battery defects that are unseen to basic voltage testing (currently used in many large battery manufacturers) can be spotted with their EchoStat technology. During this 12-week internship, I was tasked to redesign the company's database to better-handle queries and sorts, as well as other data cleaning/pipelining tools. For example, one of my full-stack tools was built to prevent the lab testers from having to clean their own data, and prevent the signal processing scientists from cleaning the lab tester's data. This was build with React.js, Python (Flask), and Sqlite3 (client-side)/PostgreSQL (server-side). I was also able to do some research on Object-Relational Mapping and implement that into the tool. At the top-level, this tool would output the last run test with all important fields to both the tester and analyist, then allow an editable state for either one to make a change (hence the lack of complexity, so that the testers didn't have to memorize long SQL queries!). Overall, this internship was a great experience, and I learned so much from my mentors.
